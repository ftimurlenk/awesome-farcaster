# Contributing to Awesome Farcaster

Thank you for helping make this list better! 🟣

## How to Add a Resource

1. **Fork** this repository
2. **Add** your resource to the appropriate section in `README.md`
3. Use this format: `[Name](url) — Short, clear description.`
4. **Submit a pull request** with a brief explanation of what you're adding and why it belongs

## Guidelines

- The resource must be **working** and publicly accessible
- The resource must be **genuinely useful** to Farcaster builders or users
- Do **not** add resources that are already listed
- Keep descriptions **concise** — one sentence max
- Alphabetical order within sections is preferred but not required

## Categories

If your resource doesn't fit an existing category, feel free to propose a new one in your PR.

## Code of Conduct

Be kind and constructive. This is a community resource for everyone.
